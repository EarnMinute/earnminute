# EarnMinute — (Post Escrow Integration)

---

# Project Overview

**EarnMinute** is a full-stack micro-task freelance marketplace where employers post small tasks and freelancers complete them for payment.

The platform is designed around:

* Fast task execution
* Clear lifecycle control
* Secure escrow-based payments
* Transparent activity and dispute tracking

Typical task categories:

* Data entry
* Social media actions
* Content writing
* Research tasks
* Design micro-jobs
* Website testing

---

# Core System Philosophy

The platform is built on **strict workflow enforcement**:

```
No action without valid state
No work without secured payment
No disputes without traceability
```

# Task Lifecycle (STRICT STATE MACHINE)

```
open
 ↓
assigned
 ↓ (escrow must be funded)
in_progress
 ↓
submitted
 ↓
revision_requested (optional)
 ↓
approved
 ↓
completed
```

---

## Alternative Flows

```
open → cancelled
assigned → cancelled
submitted → disputed
```

---

# Escrow Workflow (NOW IMPLEMENTED)

```
1 Employer posts task
2 Freelancer assigned
3 Escrow created (pending)
4 Employer funds escrow
5 Freelancer starts task
6 Freelancer submits work
7 Employer approves
8 Escrow released
9 Task completed
```

---

## Key Rule

```
Task cannot start unless escrow.status === "funded"
```
