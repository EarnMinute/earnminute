# Project: EarnMinute

EarnMinute is a **freelancing marketplace platform** similar to Fiverr/Upwork but focused on **small quick tasks** that can be completed quickly.

Employers post short tasks and freelancers apply to complete them.

The platform supports three main user roles:

* Freelancer
* Employer
* Admin

The goal is to create a **secure, scalable, production-ready marketplace**.

# Analytics System

EarnMinute includes a **custom analytics tracking system**.

Tracks daily:

```
Website visits
User registrations
User logins
Tasks created per day
```

Charts show **last 30 days of data**.

Missing days automatically display:

```
0 activity
```

Visits are counted as:

```
1 visit per IP per day
```
